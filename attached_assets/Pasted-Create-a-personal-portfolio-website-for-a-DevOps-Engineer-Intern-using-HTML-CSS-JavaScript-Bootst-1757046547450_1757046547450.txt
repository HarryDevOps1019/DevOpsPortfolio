Create a personal portfolio website for a DevOps Engineer Intern using HTML, CSS, JavaScript, Bootstrap 5, and Firebase.

❌ Do not use React, Angular, Vue, or any external libraries.
✅ Only use Bootstrap 5 + vanilla JavaScript + Firebase SDK.

Pages & Sections:

Navbar: Bootstrap 5 dark-themed sticky navbar with smooth scrolling

Home Section: Hero with name, role (DevOps Engineer Intern), and CTA button (“View My Work”)

About Section: Profile image + short bio + list of DevOps skills (Docker, Kubernetes, Terraform, AWS, Jenkins, GitHub Actions, Linux, CI/CD, Monitoring)

Skills Section: Bootstrap grid with progress bars or skill badges

Projects Section:

Load project data dynamically from Firebase Firestore

Each project card includes: title, description, tools used, GitHub link, live demo link

Resume Section: Downloadable PDF stored on Firebase Hosting (button to download)

Contact Section:

Contact form with Name, Email, Message

Save form data into Firebase Firestore

Show Bootstrap alert (success/error) after submission

Technical Requirements:

Use Firebase SDK v9+ (modular) for Firestore & Hosting

Bootstrap 5 grid, utilities, and components for layout

Vanilla JavaScript for Firebase CRUD operations

Responsive design (mobile-first)

Smooth animations for hover & scroll

Deploy with Firebase Hosting

GitHub Dark Theme Styling:

Background: #0d1117 (primary dark), #161b22 (section contrast)

Text: #c9d1d9 (light gray), headings #f0f6fc (white)

Links/Buttons: #58a6ff (blue), hover #1f6feb

Accents: #2ea043 (green for success), #f85149 (red for danger)

Cards: Rounded corners, subtle border #30363d, shadow on hover
